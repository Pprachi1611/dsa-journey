# HashMap Pattern

## When to Use

Use HashMap when:

- Fast lookup is required.
- Need to remember previously seen elements.
- Need frequency counting.
- Need complement search.
- Need mapping between two values.

---

# Common Operations

put(key, value)

Insert a key-value pair.

get(key)

Retrieve value.

containsKey(key)

Check whether key exists.

remove(key)

Delete key.
